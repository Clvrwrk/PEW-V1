## Rules for Vendors

1. **Coated stock** (glossy, satin, UV-coated): Use the **Coated (C)** Pantone number.
2. **Uncoated stock** (matte, bond, cotton): Use the **Uncoated (U)** Pantone number.
3. **Digital print / CMYK process**: Use the CMYK values. Provide the Pantone as a visual reference.
4. **Screen printing / pad printing**: Mix ink to the Coated Pantone number.
5. **Embroidery**: Reference the Coated Pantone; vendor matches closest thread.
6. **Always request a pre-production proof** before any run exceeding 100 units.

## Do Not

- Do not substitute a different Pantone number without written approval.
- Do not use the Coated number on uncoated stock (or vice versa).
- Do not convert Pantone to CMYK yourself — use the CMYK values provided above.
