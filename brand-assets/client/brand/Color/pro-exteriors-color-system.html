3. Medium-Specific Usage Guide
Different reproduction methods require different color values. This table specifies which format to use for each medium and what to watch for.
Medium Color System Notes
Website / Digital HEX, RGB, HSL Use the CSS custom properties or Tailwind tokens. sRGB color space. Always test on both calibrated monitors and typical consumer displays.
Offset Print (Coated) Pantone C + CMYK Specify Pantone for spot color jobs (business cards, letterhead, envelopes). Fall back to CMYK for 4-color process. Request a contract proof on the actual stock.
Offset Print (Uncoated) Pantone U + CMYK Colors will appear warmer and more muted on uncoated stock. The Uncoated Pantone values above account for this. Never use the Coated number on uncoated paper.
Digital Print (Toner/Inkjet) CMYK or RGB Most digital presses accept RGB and convert internally. Provide CMYK for color-critical jobs. Wide-format inkjet (banners, wraps) often uses extended gamut — provide both Pantone reference and CMYK.
Newspaper / Newsprint CMYK (adjusted) Newsprint absorbs ink heavily. Reduce total ink coverage by 15–20%. Prussian Blue and Brown Red will darken; Golden Orange will muddy. Simplify to 2–3 colors max. Test with the publication's prepress department.
Magazine (Coated) Pantone C or CMYK Magazine stock is typically coated. Use Coated Pantone values. For process color ads, supply CMYK with an ICC profile (GRACoL 2013 or publication-specific).
Direct Mail CMYK or Pantone C/U Stock varies widely. Confirm coating with your mail house. Postcards are typically coated; self-mailers may be uncoated. Match accordingly.
Embroidery Pantone (Textile) Thread color matching uses Pantone TPX/TCX (textile). Provide the Coated Pantone as a reference and let the embroiderer pull the closest thread match. Expect slight variance — thread is inherently matte.
Screen Printing (Apparel) Pantone C (ink mixing) Screen printers mix PMS inks directly. Provide the Coated Pantone numbers. On dark garments, a white underbase is required for Golden Orange and Smart Blue.
Vinyl / Vehicle Wraps Pantone C + CMYK Wrap printers use wide-format CMYK inkjet. Provide Pantone as the visual target and CMYK for the file. Laminate finish (gloss vs. matte) will shift perceived color.
Promotional Items (Mugs, Pens, etc.) Pantone C Promo vendors universally accept PMS numbers. Specify the Coated Pantone. Ceramic mugs fire at high temperatures — expect slight warmth shift. Request a pre-production sample.
Signage (Channel Letters, Monument) Pantone C + RGB Painted signs use PMS for paint mixing. Illuminated signs use LED modules — provide RGB for programmable LED. Backlit translucent faces will appear more saturated.
Hard Hats / Safety Gear Pantone C Pad printing and screen printing on plastics. PMS numbers are standard. Note: OSHA requires specific high-visibility colors for safety — ensure brand colors are applied only to non-safety-critical elements.

Restricted Pairings
Never

Gold text on White background

1.4:1 ratio. Fails all WCAG levels. Gold is invisible on white for visually impaired users.

Never

Blue text on Navy background

1.9:1 ratio. Fails all WCAG levels. Virtually indistinguishable for most users.

Never

Green text on Red background (or reverse)

1.2:1 ratio. Fails WCAG and is invisible to red-green colorblind users (8% of males).

Never

Gold on Green / Green on Gold

2.9:1 ratio. Fails all WCAG levels including large text. These colors are too close in luminance.

Never

Gold on Blue / Blue on Gold

2.6:1 ratio. Fails all WCAG levels. Do not pair these for any text or meaningful UI elements.

Never

Red on Gold / Gold on Red

2.7:1 ratio. Fails all WCAG levels. The brighter red and gold are too close in luminance to pair for any text.

7. Color Vision Deficiency Reference
How the palette appears under the most common forms of color blindness. Data from the Coolors simulation. The palette maintains sufficient differentiation across all conditions because it relies on value contrast (light vs. dark) rather than hue alone.
Condition % Affected Green Red Navy Gold Blue Design Impact
Normal Vision — —
Protanopia ~1% males Green & Red converge in hue. Value contrast preserved.
Deuteranopia ~6% males Most common CVD. Green/Red shift but remain separable by lightness.
Tritanopia ~0.01% Gold shifts to pink. Navy shifts teal. Rare condition.
Achromatopsia ~0.003% Full grayscale. Navy/Gold maintain strong value separation. Green/Red/Blue converge — never rely on these three alone to convey different states.
